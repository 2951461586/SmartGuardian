# 学生智慧托管系统字段字典

> 项目名称：基于鸿蒙系统的学生智慧托管系统  
> 文档类型：字段字典  
> 版本：V1.2

## 一、说明

本文根据增强版建表 SQL 提炼核心业务表字段，供开发、联调与测试参考。完整字段以 `数据库建表SQL-增强版.sql` 为准。

## 二、核心表字段摘要

### 2.1 user
| 字段 | 类型 | 说明 |
|---|---|---|
| id | BIGINT | 主键ID |
| account_no | VARCHAR(64) | 账号编号 |
| username | VARCHAR(64) | 登录名 |
| password_hash | VARCHAR(255) | 密码哈希 |
| real_name | VARCHAR(64) | 真实姓名 |
| mobile | VARCHAR(32) | 手机号 |
| role_type | VARCHAR(32) | 角色类型 |
| status | VARCHAR(16) | 状态 |

## 三、公共审计字段

多数增强版表均包含以下字段：
- is_deleted：逻辑删除标记
- created_by：创建人
- updated_by：更新人
- created_at：创建时间
- updated_at：更新时间
