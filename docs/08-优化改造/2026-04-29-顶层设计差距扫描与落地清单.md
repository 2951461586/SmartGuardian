## 本批已落地

1. AGC 初始化兜底：
   - `AgcRuntimeService` 在 `resourceManager` 不可用时给出明确 Previewer/设备环境提示。
   - `LoginPage` 在 AGC client config 未 ready 时停止继续调用云函数。

2. 考勤/请假能力补齐：
   - 前端新增 `AttendanceService.getLeaveRequests()`、`cancelLeave()`、`getStatistics()`。
   - 云函数新增 `GET /api/v1/attendance/statistics`。
   - 云函数新增 `GET /api/v1/attendance/leave`。
   - 云函数新增 `POST /api/v1/attendance/leave/{leaveId}/cancel`。

3. 班次路由统一：
   - 前端 `SessionService.getTodaySessions()` 改为调用 `/api/v1/sessions/today`。
   - 云函数新增 `GET /api/v1/sessions/today`。

4. Preview/stub 命名治理：
   - `previewMessageDispatch` 更名为 `dispatchLocalMessageNotification`。
   - 分布式相关日志从 `stub mode` 改为 `local compatibility mode`，明确当前是本地兼容层。

5. 旧 Mock 运行块清理：
   - 删除未引用的 `services/api/BaseApiService.ets`，移除 `MockControl` / `MockServiceSwitch` 旧基类。
   - `check-legacy-artifacts.ps1` 新增旧 Mock switch、preview/stub 词门禁。

6. UI emoji 门禁：
   - `check-ui-governance.ps1` 新增活跃 UI emoji literal 扫描。
   - 当前扫描结果：活跃 `pages/`、`components/` 中纯 emoji 文本为 0。
