## 本批直接落地

1. 安全审计落库：
   - 新增 `cloud-functions/shared/audit.js`。
   - `shared/router.js` 对非 GET 请求和失败请求记录 `security_audit_events`。
   - Cloud DB schema、collections、seed 和函数 manifest/contract 补齐 `security_audit_events`。

2. Agent 安全收口：
   - Agent 调用 LLM 前改用最小化上下文，不再把学生姓名、健康备注、消息正文、告警描述、订单金额等明细直接送入外部模型。
   - Agent 仍保留本地规则回答的完整业务体验，未配置 LLM key 时继续返回 `LOCAL_RULES`。

3. 告警作用域清理：
   - 新增 `filterAlertsForUser()`，替代告警域和 Agent 聚合里复用考勤过滤器的写法。
   - 行为不扩权，命名和职责更清晰，便于后续细化教师/管理员作用域。

4. 烟测补强：
   - `cloud-functions` smoke 覆盖 Agent 对话、总结、导航与审计事件。
   - 验证失败权限请求会生成失败审计记录。
