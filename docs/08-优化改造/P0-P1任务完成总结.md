### ✅ 3. 扫码页面集成权限请求

**文件**: `entry/src/main/ets/pages/teacher/TeacherScanPage.ets`

**实现功能**:
- ✅ 启动扫描前请求相机权限
- ✅ 权限拒绝时显示友好提示
- ✅ 集成ErrorHandlerUtil处理错误
- ✅ 成功/失败操作显示Toast提示

**关键代码**:
```typescript
async startScan() {
  // Request camera permission
  const context = getContext(this) as common.UIAbilityContext;
  const hasPermission = await PermissionUtil.requestCameraPermission(context);
  
  if (!hasPermission) {
    this.errorMessage = '需要相机权限才能扫描二维码';
    this.isScanning = false;
    ErrorHandlerUtil.showErrorToast('请在设置中开启相机权限');
    return;
  }
  
  await this.initCamera();
}
```
