# 2026-04-27 重构改造大颗粒执行清单

## 当前判断

- 主链路已经从旧后端收口到 `DEV_MOCK` 与 `AGC_SERVERLESS` 两种运行模式。
- 三端页面、工作台 manifest、Mock handler、OpenAPI、AGC 函数契约和 Cloud DB schema 已形成一致性检查。
- 当前优先风险不是页面功能缺口，而是公共请求层、服务契约、ArkTS 语法约束和官方 Kit 接入边界的持续收口。
