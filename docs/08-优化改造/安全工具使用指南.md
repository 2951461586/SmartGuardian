# SmartGuardian 安全工具使用指南

## 1. 数据脱敏工具 (DataMaskUtil)

### 引入
```typescript
import { DataMaskUtil } from '../utils/DataMaskUtil';
```

### 常用方法
```typescript
// 手机号脱敏: 138****5678
DataMaskUtil.maskPhone('13812345678');

// 姓名脱敏: 张**
DataMaskUtil.maskName('张小明');

// 身份证脱敏: 110101********1234
DataMaskUtil.maskIdCard('110101199001011234');

// 邮箱脱敏: te***@example.com
DataMaskUtil.maskEmail('testuser@example.com');
```

### 页面中使用
```typescript
Text(DataMaskUtil.maskPhone(student.phone))
```

## 2. 加密工具 (CryptoUtil)

### 引入
```typescript
import { CryptoUtil } from '../utils/CryptoUtil';
```

### 加密解密
```typescript
// 加密
const encrypted = CryptoUtil.encryptSimple('敏感数据');

// 解密
const decrypted = CryptoUtil.decryptSimple(encrypted);
```

### 密码处理
```typescript
// 密码哈希
const hashed = await CryptoUtil.hashPassword('password123');

// 密码验证
const isValid = await CryptoUtil.verifyPassword('password123', hashed);
```

### 其他功能
```typescript
// Base64编解码
const encoded = CryptoUtil.base64Encode('Hello');
const decoded = CryptoUtil.base64Decode(encoded);

// Token生成
const token = CryptoUtil.generateToken();

// 随机字符串
const randomStr = CryptoUtil.generateRandomString(16);
```

## 3. 隐私协议弹窗 (PrivacyDialog)

### 引入
```typescript
import { PrivacyDialog, isPrivacyAgreed } from '../components/PrivacyDialog';
```

### 使用示例
```typescript
@Entry
@Component
struct LoginPage {
  privacyDialogController: CustomDialogController = new CustomDialogController({
    builder: PrivacyDialog({
      onAgree: () => {
        console.log('用户同意');
      },
      onDisagree: () => {
        console.log('用户拒绝');
      }
    }),
    autoCancel: false,
    customStyle: true
  });

  aboutToAppear() {
    if (!isPrivacyAgreed()) {
      this.privacyDialogController.open();
    }
  }
}
```

## 5. 最佳实践

### 5.1 敏感数据处理
```typescript
// 前端展示时脱敏
Text(DataMaskUtil.maskPhone(user.phone))

// 传输前加密
const encrypted = CryptoUtil.encryptForTransmission(data);

// 存储前加密
const hashed = await CryptoUtil.hashPassword(password);
```

### 5.2 安全登录流程
```typescript
async function handleLogin() {
  // 1. 检查隐私协议
  if (!isPrivacyAgreed()) {
    this.privacyDialogController.open();
    return;
  }

  // 2. 密码哈希
  const hashedPassword = await CryptoUtil.hashPassword(password);

  // 3.登录
  const resp = await AuthService.login({
    username,
    password: hashedPassword
  });
}
```

## 6. 注意事项

1. 数据脱敏只用于前端展示，不影响原始数据
2. 加密密钥要妥善保管，生产环境建议使用系统加密API
3. 隐私协议必须在收集数据前获得用户同意
4. 扫码功能需要相机权限
