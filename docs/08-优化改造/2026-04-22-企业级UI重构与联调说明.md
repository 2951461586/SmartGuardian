# 2026-04-22 UI 重构与本地联调说明

## 1. 本轮目标

这一轮改造把项目从“远端部署联调”切回“本地构建联调”，并同步重做前端工作台视觉：

- 后端统一为本地 Spring Boot + MySQL
- 前端真实环境默认地址改为 `http://127.0.0.1:8080`
- 登录页、主工作台和共享组件统一升级视觉语言

## 4. UI 方向

本次前端不再沿用通用浅蓝企业模板，而是切到一套更有辨识度的“暖纸张 + 深墨色 + 玉石绿”工作台风格：

- 背景更柔和，减少默认蓝白面板感
- 顶部欢迎区和底部导航都改为更有层次的容器
- 卡片、快捷入口、状态统计统一使用更温润的边框和阴影
- 登录页增加本地联调语义，避免继续把服务器地址当默认入口
