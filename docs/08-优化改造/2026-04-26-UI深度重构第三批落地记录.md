## 本轮落地

### 1. 报表页模板化

- 页面：
  - `entry/src/main/ets/pages/admin/AdminReportsPage.ets`
- 新增统一报表分段组件：
  - `entry/src/main/ets/components/common/AdminReportSections.ets`

### 2. 家长首页二次拆分

- 页面：
  - `entry/src/main/ets/pages/parent/ParentHomePage.ets`
- 新增统一首页工作台段：
  - `entry/src/main/ets/components/common/ParentHomeSections.ets`
- 本轮收口内容：
  - 首页信息块回收进 `ParentHomeWorkspaceSection`
  - 页面层只负责容器、响应式分栏和数据装配
  - 降低首页继续堆叠视觉细节和业务段落的风险

### 3. 家长退款页继续收口

- 页面：
  - `entry/src/main/ets/pages/parent/ParentRefundPage.ets`
- 新增退款分段组件：
  - `entry/src/main/ets/components/common/ParentRefundSections.ets`
- 本轮收口内容：
  - 退款原因区抽成 `RefundReasonSection`
  - 退款记录区抽成 `RefundHistorySection`
  - 页面统一接入 `PageScaffold + StatefulContainer + FormActionBar`

### 4. 管理端学生/订单区块令牌化

- 区块组件：
  - `entry/src/main/ets/components/common/AdminStudentSections.ets`
  - `entry/src/main/ets/components/common/AdminOrderSections.ets`
- 本轮收口内容：
  - 保持原有导出组件接口不变，避免页面联动回归
  - 硬编码颜色、阴影和边框改为 `HarmonyColor`、`HarmonyReport`、`harmonyShadowColor`
  - 统一卡片留白、圆角和操作区节奏

### 5. 模板和资料页底座延续复用

- 模板补强：
  - `entry/src/main/ets/components/common/SceneTemplates.ets`
  - `entry/src/main/ets/components/common/ProfileCenterSections.ets`
