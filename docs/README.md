## 8. 使用建议

1. 评审与答辩优先阅读项目概览、顶层设计方案、数据库设计说明书。  
2. 联调优先查看 API 清单、接口示例、OpenAPI 原始文件。  
3. 部署实施优先查看部署手册、上线检查清单、运维巡检手册。  
4. 导出 PDF/Word 前，建议参考《文档编写与排版规范》统一页眉页脚与版式。
