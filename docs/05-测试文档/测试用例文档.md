## 1. 测试范围

覆盖登录、学生档案、订单管理、签到签退、作业反馈、消息通知、报表统计等核心业务模块。

### 1.1 当前测试基线（2026-04-27）

- 当前默认环境为 `DEV_MOCK`，位于 `entry/src/main/ets/config/api.config.ts`。
- Mock 与 AGC 契约一致性已纳入 `scripts/check-agc-consistency.ps1`。
- AGC Serverless 路径使用 Cloud Functions smoke、OpenAPI、函数 manifest 和 Cloud DB schema 进行闭环验证。
- 本地真机签名 Profile 需要连接 USB/IP 设备后生成；模拟器安装 HAP 可暂时跳过签名。

## 2. 典型测试用例

| 编号 | 模块 | 用例名称 | 前置条件 | 预期结果 |
|---|---|---|---|---|
| TC-001 | 登录 | 家长正常登录 | 账号存在且密码正确 | 登录成功，返回 Token |
| TC-002 | 学生档案 | 查询学生详情 | 家长与学生已绑定 | 返回学生档案 |
| TC-003 | 订单 | 创建托管订单 | 学生和服务存在 | 订单创建成功 |
| TC-004 | 订单 | 审核通过订单 | 管理员已登录 | 订单状态更新为已审核 |
| TC-005 | 签到 | 正常签到 | 班次有效、时间窗有效 | 状态更新为已签到 |
| TC-006 | 签到 | 重复签到拦截 | 已存在签到记录 | 返回重复签到错误 |
| TC-007 | 签到 | 非法时间窗签到 | 超出签到时间范围 | 触发异常提醒 |
| TC-008 | 作业反馈 | 教师提交反馈 | 作业任务存在 | 反馈提交成功 |
| TC-009 | 家长反馈 | 家长确认反馈 | 存在未确认反馈 | 状态更新为已确认 |
| TC-010 | 消息 | 发送站内消息 | 双方账号有效 | 消息发送成功 |

## 3. 性能测试关注点

- 高峰期签到并发
- 首页与卡片摘要加载时间
- 订单列表分页查询响应时间
- 报表聚合接口响应时间

## 4. 安全测试关注点

- 未授权访问拦截
- 敏感字段脱敏展示
- HTTPS 传输加密验证
- Token 过期与越权测试
