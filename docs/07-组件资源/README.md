# 组件资源目录说明

> 文档类型：资源索引  
> 日期：2026-04-27

本目录保存设计资源、字体、图标、原型图和第三方资源许可，不作为业务说明文档维护。

## 资源分类

- `HarmonyOS+Sans+字体/`：HarmonyOS Sans 字体资源及授权说明。
- `华为分享图标/`：华为分享相关图标、PNG、SVG、Sketch、Pixso 源文件。
- `使用华为账号登录/`：华为账号登录 logo 与加载动画资源。
- `手机折叠屏平板/`：HarmonyOS 组件库设计源文件。
- `项目原型图/`：家长端、教师端、机构端、登录注册等原型图。
- `元服务静默登录默认头像/`：元服务默认头像 PNG / SVG。
- `LUCIDE-ICON-LICENSE.txt`：Lucide 图标许可。

## 维护规则

- 设计源文件和位图资源保留原始文件名。
- 资源进入应用代码前，应复制到 `entry/src/main/resources/` 并按 HarmonyOS 资源命名规范重命名。
- 第三方资源必须保留 license 或来源说明。
