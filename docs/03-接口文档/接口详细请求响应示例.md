## 一、登录接口示例

### 请求
`POST /api/v1/auth/login`

```json
{
  "username": "parent_zhang",
  "password": "123456"
}
```

### 响应
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "token": "Bearer xxx.yyy.zzz",
    "userId": 10001,
    "roleType": "PARENT",
    "realName": "张女士"
  }
}
```

## 二、创建订单示例

### 请求
`POST /api/v1/orders`

```json
{
  "studentId": 20001,
  "serviceProductId": 30001,
  "startDate": "2026-04-17",
  "endDate": "2026-06-30",
  "remark": "需要晚托辅导作业"
}
```

### 响应
```json
{
  "code": 200,
  "message": "创建成功",
  "data": {
    "orderId": 40001,
    "orderNo": "SG202604160001",
    "orderStatus": "CREATED",
    "payStatus": "UNPAID"
  }
}
```

## 四、作业反馈示例

### 请求
`POST /api/v1/homework/feedback`

```json
{
  "homeworkTaskId": 70001,
  "studentId": 20001,
  "teacherUserId": 10021,
  "progressRate": 90,
  "feedbackText": "数学作业已基本完成，错题已订正。",
  "performanceTags": ["专注", "积极"]
}
```

### 响应
```json
{
  "code": 200,
  "message": "提交成功",
  "data": {
    "feedbackId": 80001,
    "parentConfirmStatus": "UNCONFIRMED"
  }
}
```
