# 学生智慧托管系统部署手册

> 项目名称：基于鸿蒙系统的学生智慧托管系统  
> 文档类型：部署手册  
> 版本：V1.2  
> 日期：2026-04-16

---

## 一、部署环境

- 开发工具：DevEco Studio
- 前端技术：ArkTS / ArkUI
- 后端环境：Java/Node 等 REST 服务（按实际实现选型）
- 数据库：MySQL 8.0+
- 缓存：Redis
- 消息队列：MQ

## 二、部署内容

1. 导入数据库建表脚本。  
2. 执行初始化测试数据脚本。  
3. 部署后端 API 服务。  
4. 配置 Redis 与 MQ。  
5. 导入 OpenAPI 文档供联调。  
6. 使用 DevEco Studio 编译并安装 HarmonyOS 客户端。

## 三、数据库初始化

建议执行顺序：
1. `数据库建表SQL.sql` 或 `数据库建表SQL-增强版.sql`
2. `初始化测试数据.sql`

## 四、配置建议

- 区分开发、测试、生产环境配置。  
- 统一管理数据库、缓存、MQ 与对象存储连接信息。  
- 生产环境开启 HTTPS、日志审计与备份机制。

## 五、上线前检查

- 核心接口可访问
- 数据库连接正常
- Redis 与 MQ 正常
- 登录、下单、签到、反馈主流程可用
- 日志、告警与监控可用
